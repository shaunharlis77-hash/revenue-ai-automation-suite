# Project Charter

## Purpose

The revenue AI automation suite exists to reduce repetitive sales operations work while keeping people in control of customer-facing decisions.

The project will eventually help sales and revenue teams with tasks like lead prioritization, meeting summaries, follow-up drafts, CRM cleanup, proposal support, knowledge retrieval, and impact reporting.

## Current Scope

This repository currently contains the foundation only:

- A FastAPI backend with a health endpoint.
- A Next.js dashboard with placeholder pages.
- Documentation for the intended direction.
- Folders for future SOPs, knowledge content, sample data, and n8n workflows.

## Out of Scope for the Foundation

- No real AI workflows.
- No HubSpot integration.
- No external API calls.
- No authentication.
- No real API keys.
- No production customer data.

## Success Criteria

The foundation is successful if a reviewer can understand the project, run the basic app skeletons, and see where future sales AI automation work will live.

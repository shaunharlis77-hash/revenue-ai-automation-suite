# Sales Rep Task Map

## Why This Map Exists

The project is organized around real sales tasks instead of generic AI features. This keeps automation tied to useful work.

## Task Areas

| Sales task | Future AI support | Human role |
| --- | --- | --- |
| Prioritize leads | Suggest scoring signals and rank accounts | Confirm priority and outreach strategy |
| Summarize meetings | Draft notes, next steps, risks, and objections | Correct details and approve summary |
| Write follow-ups | Draft emails based on approved context | Edit and send |
| Maintain CRM data | Flag missing, stale, or inconsistent fields | Approve changes before write-back |
| Draft proposals | Suggest structure and approved language | Own pricing, scope, and final content |
| Answer sales questions | Retrieve approved knowledge base content | Validate fit for the deal |
| Report impact | Track time saved and workflow quality | Interpret metrics and decide improvements |

## Foundation Status

The dashboard includes placeholder pages for these task areas. No task automation is active yet.
